# 视无界社区管理端

基于 Vue 3 + Ant Design Vue 的社区管理系统前端项目。

## 技术栈

- Vue 3
- Ant Design Vue
- Vue Router
- Pinia
- Axios
- Vite

## 项目结构

```
src/
├── views/                 # 页面组件
│   ├── login/            # 登录页面
│   ├── layout/           # 主布局
│   ├── dashboard/        # 仪表板
│   ├── community/        # 社区管理
│   ├── activity/         # 活动管理
│   ├── helppost/         # 求助管理
│   ├── user/             # 用户管理
│   └── statistics/       # 数据统计
├── stores/               # 状态管理
├── router/               # 路由配置
└── main.js              # 入口文件
```

## 功能模块

### 1. 社区管理
- 社区列表查看
- 社区创建/编辑/删除
- 社区加入审核

### 2. 活动管理
- 活动列表查看
- 活动创建/编辑/删除
- 活动报名管理

### 3. 求助管理
- 求助帖列表查看
- 求助帖审核/删除

### 4. 用户管理
- 志愿者管理
- 视障人士管理

### 5. 数据统计
- 社区统计
- 活动统计

## 开发环境

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 登录说明

目前使用模拟登录，输入任意手机号和密码即可登录。

## 注意事项

- 所有数据都是模拟数据，实际使用时需要对接后端API
- 图标使用emoji字符，避免资源加载问题
- 页面功能框架已搭建完成，等待后续API对接
