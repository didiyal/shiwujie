# 导航

您想去某个地方时，我可以帮您搜索地点、规划路线，并在手机上开启导航。

## 操作步骤

1. 告诉我「导航去 XX」或「带我去最近的医院」。
2. 我会搜索附近的候选地点，若有多个会读给您选。
3. 我会问您步行、公交还是驾车。
4. 选好后我规划路线、朗读距离和时间，并在手机上开启导航。

## 注意

- 目的地请尽量说清楚（如「最近的医院」）；模糊时我会先搜候选让您选。
- 中途想取消，告诉我即可。
