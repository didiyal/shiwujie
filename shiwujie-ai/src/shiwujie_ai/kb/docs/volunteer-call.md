# 志愿者视频通话

当您需要真人当眼睛、但**不是紧急情况**时（看不清路牌、不会操作某功能等），呼叫志愿者视频帮扶。

## 操作步骤

1. 告诉我「找个志愿者」或「视频求助」。
2. 我会从志愿者队列匹配一位最近的可用志愿者。
3. 接通后志愿者可通过视频帮您看清周围、指导操作。

## 注意

- 真正的紧急（受伤、疾病、遇险）请用「紧急求助」通知家属，不要用本功能。
- 志愿者按先后顺序匹配，可能需要稍等。
