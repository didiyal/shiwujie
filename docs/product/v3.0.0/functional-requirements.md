# 功能需求（v3.0.0 · 进行中）

> 单体化改造版本。用户可见功能需求**继承 v2.1.0**（完整 `FR-<MODULE>-<NN>` 清单见 [../v2.1.0/functional-requirements.md](../v2.1.0/functional-requirements.md)，已封版 2026-07-11）。本版本聚焦单体化重构，**原则上不改用户可见业务功能**；因单体化可能产生的契约变更（如入口/端口收敛）待架构定稿后在此补 FR 编号。

## 继承基线

AI 助手 / 视频帮扶 / 紧急求助 / 社区治理 / 用户家庭的全部 `FR-*` 继承自 v2.1.0，不在此重复（去重铁律：FR/AC 只在 product 出现一次）。

## v3.0.0 新增 / 变更

### 社区域权限边界（安全加固，2026-07-12）

- **FR-COMMUNITY-PERM-01**：平台须对求助帖删除/修改、社区修改/删除、社区管理员增删改执行授权校验——求助帖删改限作者或本社区注册人/管理员；社区修改/删除仅限注册人；社区管理员管理仅限注册人/管理员（员工无权），并保护唯一注册人不被降级/删除、不新增第二个注册人。越权请求返回业务码 `NO_AUTH`（40030）。

### 用户口令存储（安全加固，2026-07-12）

- **FR-AUTH-PWD-01**：用户口令须以 BCrypt 加盐哈希存储，不以无盐 MD5 存储；存量无盐哈希在首次正确登录时自动升级。
